MiddleKit 0.2.1 Release Notes

Webware for Python 0.7 beta 1

Changes

  • Version 0.2.1 was released on 12/10/2001.
  • In certain situations, MiddleKit could store an invalid object reference in the SQL database. This would cause a warning when retrieving the object and a disconnect between the source and target objects.
  • MiddleKit didn't properly support the 'Package' setting in many cases.
