MiddleKit 0.7 Release Notes

Webware for Python 0.7

Changes

  • New versioning strategy: all Webware components will now have their version numbers updated in lock-step.
  • A new isIndexed=1 flag can be set for each attribute, which results in a SQL index. This is also for speed up especially when used on "list back references".
  • store.deleteObject() is now an order of magnitude faster.
  • store.fetchObjectsOfKlass() is now over twice as fast.
