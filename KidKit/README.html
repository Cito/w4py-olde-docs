KidKit ReadMe

Created by : Winston Wolff January 2005

 

KidKit is a Webware plugin to which makes Webware understand *.kid files. By installing this kit, you can write .kid template files and Webware will render them automatically without you needing to compile them.

It is in ALPHA state I would say at the moment (January 2005). I put it together and it works for some quick tests. You can access the put and field variables, and it does for loops and so forth. If there is much demand,

 

Requirements

Kid, Element-Tree and Webware. This was built and tested in January 2005, with Kid v0.4.1 and Element Tree 1.2.4-20041228. Other versions may work.

  • Kid by Ryan Tomayko
  • Element-Tree by Fredrik Lundh
  • Webware CVS version from January 2005

 

Installation

  • Install Kid and add it to your python path.
  • Install Element-Tree and add it to your python path.
  • Copy the samples folder to your webware context and try loading some of the files.
