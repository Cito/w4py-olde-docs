Synopsis

KidKit is a Webware plugin to which makes Webware understand *.kid files. By installing this kit, you can write .kid template files and Webware will render them automatically without you needing to compile them.

It is in alpha state I would say at the moment (January 2005). I put it together and it works for some quick tests. You can access the httmp GET/PUT field variables, and it does "for" loops. If you use Kid a lot, feel free to improve this component.

Requirements

Kid, Element-Tree and Webware.

This was built and tested in January 2005, with Kid v0.4.1 and Element Tree 1.2.4-20041228. Other versions may work.
