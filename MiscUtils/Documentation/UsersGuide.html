MiscUtils User's Guide

Version 0.4
Webware for Python 0.4

Synopsis

MiscUtils provides support classes and functions to Webware that aren't necessarily web-related and that don't fit into one of the other frameworks.

Feedback

You can e-mail webware-discuss@lists.sourceforge.net to give feedback, discuss features and get help using MiscUtils.

Classes

There two interesting classes contained in MiscUtils, each of which contains it's own documentation embedded in the source. In future releases, we'll extract the docs so they can be read independently.

DataTable.py
NamedValueAccess.py

Known Bugs

Known bugs and future work in general is documented with each class.

Also, in the next release of Webware, the CSV class, currently used for logging by WebKit, will be subsumed by DataTable.

Also, WebKit.OneShotAdapter's WordWrap() function should be moved here.
