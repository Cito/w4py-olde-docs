MiscUtils 0.8 Release Notes

Version 0.8
Webware for Python 0.8

General
  • New PickleCache module which is now used by DataTable and MiddleKit.Core.Model in order to load files more quickly. See the doc string for details.
  • New MiscUtils.Funcs.wordWrap(s, width=78) function which does just what you would expect.
  • New MiscUtils.Funcs.localIP(remote=('www.yahoo.com', 80), useCache=1) function which returns your local IP address, hopefully as seen by the Internet. It comes with caveats, so read the doc string.
