WebUtils 0.9.2 Release Notes

WebUtils version 0.9.2 released on 09/18/06

Improvements and Refinements

  • urlDecode() and urlEncode() in WebUtils.Funcs now use the same implementation as quote_plus() and unquote_plus() in urllib of Python 2.4, which is faster than in older Python and Webware versions. This also means that urlDecode() now silently ignores erroneous encoding (as had been suggested by Ben Parker), including codes with only one digit.
