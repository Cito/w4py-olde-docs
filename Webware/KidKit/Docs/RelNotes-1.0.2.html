KidKit 1.0.2 Release Notes

KidKit version 1.0.2 released on 06/07/09

New Features

  • The file extensions of the files that shall be parsed as Kid templates can now be configured using the setting ExtensionsForKid. By default (as before), only files with the extension .kid are regarded as Kid templates.
