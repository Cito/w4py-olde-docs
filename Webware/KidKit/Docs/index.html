KidKit Documentation

Name: KidKit Summary:
Version: 1.1.1 KidKit is a Webware plug-in that allows Kid templates to be automatically compiled and run as servlets by the WebKit application server. Kid is a simple templating language for XML based vocabularies written in Python. You need to install the Kid package before you can use the KidKit plug-in.
Status: beta
Py Ver: 2.4
Webware: 1.1.1
