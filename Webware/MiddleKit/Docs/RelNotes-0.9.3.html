MiddleKit 0.9.3 Release Notes

MiddleKit version 0.9.3 released on 04/26/07

Improvements and Refinements

  • Ensured backward compatibility with older Python versions.
  • Made store.object() more flexible: it can take an ObjectKey or a separate class and serial num.
  • Removed the timestamp from the Python code generator to reduce source code control diffs. There is still a timestamp in Info.text.
