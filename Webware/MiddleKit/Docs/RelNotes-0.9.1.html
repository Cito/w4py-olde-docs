MiddleKit 0.9.1 Release Notes

MiddleKit version 0.9.1 released on 04/23/06

New Features

  • The SQLObjectStore class has got a new method refreshObject().

Bugfixes

  • You now get a useful error message when the class ids in the database don't match the class names in the model.
  • A bug with passing the database argument and a couple of other bugs have been fixed.
